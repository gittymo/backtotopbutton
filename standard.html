A Back To Top Button Using Standard HTML Elements

This is a demonstration of a back to top button using standard HTML elements and a bit of JavaScript.

The button remains hidden until the user scrolls down 60% of the page height. Once visible, clicking the button smoothly scrolls the page back to the top.

Whilst this approach is simple and effective, it has a few limitations, such as the scroll percentage only being changeable in the JavaScript code, which means the user cannot adjust it without modifying the code and being aware of what the code does.

The additional JavaScript coding requirements make implementation a little harder, but it does allow for more customization and control over the button's behavior than a basic static button.
